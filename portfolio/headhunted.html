Headhunted — Website UI & Frontend Design

I designed the UI and frontend-ready layout for Headhunted, a nonprofit initiative supporting education and aid programs for children, single mothers, and individuals in need. The goal was to deliver a clear, trustworthy website experience that encourages community awareness and engagement.

Category Nonprofit · Web
Role Freelance Product Designer
Collaboration Smarttie (Development Partner)

Overview

Headhunted needed a website that communicated credibility, purpose, and clarity — while remaining simple to navigate and easy to maintain. The project included brand direction refinements, page structure, and UI design aligned to accessibility and readability best practices.

Context

I partnered with Smarttie (development agency) during the design and build phase. My role focused on designing the interface system — from layout and navigation to component specifications — ensuring the implementation could ship without ambiguity.

Goals

  • Create a visual identity that feels trustworthy, modern, and mission-driven.
  • Design a readable, intuitive website that guides users through the purpose and ways to help.
  • Deliver frontend-ready UI with clear specs for efficient implementation.

Research & Discovery

The discovery phase combined lightweight market research and stakeholder alignment. We clarified the audience, content priorities, and the tone required to communicate impact without feeling transactional.

  • Mapped key audiences and primary user intents (learn, trust, participate, contact).
  • Reviewed comparable nonprofit websites to identify content patterns that increase credibility.
  • Validated information hierarchy to prioritize mission, programs, and calls to action.

Brand Direction

The brand direction aimed to balance warmth with professionalism — pairing a clean typographic system with restrained color usage and generous whitespace. The result supports readability and reinforces credibility.

UI Design & Component System

I translated the brand into a consistent UI system: buttons, forms, content blocks, and responsive layouts. Each component was designed to remain legible across devices and easy to implement within typical web constraints.

  • Designed responsive sections for storytelling (mission, programs, outcomes, and ways to help).
  • Defined component states for interactive elements (hover, focus, error).
  • Ensured accessibility basics: contrast, hierarchy, and predictable navigation patterns.

Collaboration with Smarttie

To ensure a smooth handoff, I provided annotated designs, component behaviors, and spacing rules, reducing back-and-forth during implementation. We aligned on constraints early to avoid redesign late in development.

  • Shared design specs and assets with clear naming conventions.
  • Validated implementation details through iterative reviews.
  • Supported QA feedback to keep UI consistent across breakpoints.

Outcome

The final website delivered a clear narrative, strong credibility signals, and a consistent UI system that supports future growth. The design enables users to quickly understand the mission, explore initiatives, and take action without friction.

Next Steps

If extended, I would add lightweight analytics to validate content engagement, refine calls to action based on behavior, and iterate on SEO fundamentals to improve discoverability for the communities served.
